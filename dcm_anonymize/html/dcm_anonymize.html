dcm_anonymize.anonimization.add_not_dcm(files, pathout, parent_folder)[source]

Add dicom files to the list of files to be anonimized also if they have no extension .dcm. .mim files are excluded and copied in the new destination as they are. :param files: list of file names found with .dcm (str list). :param pathout: path to the folder to which the file will be saved (str). :param parent_folder: folder where data to be anonimized are (str). :return: files: list of file names found (str list). :return: unique_folder: Folders from which files were imported (selected to be unique).

dcm_anonymize.anonimization.anonimize(parent_folder, modality, list_fields=None, path=None, parallel=True, mim=True)[source]

Anonimize all the dicom files inside a specific folder. Note: if none of the files are .dcm in a single folder, one of them have to be modified previously. For this it is needed to be known the folder’s name. :param parent_folder: folder where data to be anonimized are (str). :param modality: :param list_fields: list of keyword to be anonimized inside the .dcm files (str list). :param path: path to be considered for reading and saving (str). :param parallel: True if parallelization is used (bool). :param mim: if the analysis is done for MIM software or not. :return: 0

dcm_anonymize.anonimization.modifydsc(file, list, folder)[source]

Anonimize the dicom file. If the file has not the dicom header it will be forced to be read anyway. In place of ‘PatientName’ and ‘PatientID’ it will be written the patient folder ID. :param file: single file name (str). :param list: list of keyword to be anonimized inside the .dcm files (str list). :param folder: folder where the file is originally located (str). :return: ds: dicom file read and modified.

dcm_anonymize.anonimization.savedsc(ff, fileout_path, ds)[source]

Save an anonimized copy of the file in a selected new folder. Checks are applied to reduce the file name (if the characters are too many to be saved) or to add the extension .dcm if the original ones had it implicit. :param ff: single file name (str). :param fileout_path: folder to which the file will be saved (str). :param ds: dicom file modified to be saved. :return: 0
