dcm_anonymize.anonymization.anonymize(parent_folder, modality, list_fields=None, path=None, parallel=True, mim=True)[source]

anonymize all the dicom files inside a specific folder. Note: if none of the files are .dcm in a single folder, one of them have to be modified previously. For this it is needed to be known the folder’s name.

Parameters:

  • parent_folder – folder where data to be anonymized are (str).

  • modality – modality of images e.g., ‘CT’.

  • list_fields – list of keyword to be anonymized inside the .dcm files (str list).

  • path – path to be considered for reading and saving (str).

  • parallel – True if parallelization is used (bool).

  • mim – if the analysis is done for MIM software or not.

dcm_anonymize.anonymization.modifydsc(file, list, folder)[source]

Anonimize the dicom file. If the file has not the dicom header it will be forced to be read anyway. In place of ‘PatientName’ and ‘PatientID’ it will be written the patient folder ID.

Parameters:

  • file – single file name (str).

  • list – list of keyword to be anonymized inside the .dcm files (str list).

  • folder – folder where the file is originally located (str).

Returns:

ds: dicom file read and modified.
